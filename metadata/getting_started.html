Adding extra metadata

Adding extra metadata to files in Rinobot

If you want extra metadata to be saved with files uploaded to Rinocloud just go to the 'Metadata' tab of the file setting and insert any values you wish.

You can refer to these values again in the commands, using the notation {{metadata-name}}

For example if we want to add the location of some network files to copy we just add a location to the metadata

We can reference the metadata parameter in the copy task by inserting the copy location target as {{location}}

This location will then be attached to the metadata of any uploaded files.

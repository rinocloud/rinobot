Installation

Rinobot

Rinobot is an app to automatic research data processing.

It syncs everything in your folders with rinocloud. It also lets you hook in packages to automatically process your data when it appears in the folder.

Currently you'll need a Rinocloud account to use Rinobot. This is for verification, and so that you can sync with your Rinocloud account. If your using the on-site version of Rinocloud, or don't want to put your data in the cloud; thats no problem.

Follow the installation instructions below and then head to the Getting started guide.

Installing Rinobot

  1. Download the installer below for windows or mac. You do not need administrator privileges to install Rinobot, it installs into the local user account.

    • Installer for Windows
    • Installer for Mac

  2. Open the downloaded file and allow Rinobot to set itself up!

  3. When prompted, log in using your Rinocloud account.

  4. (Optional) Plugins: Install the Anaconda python framework to use the library of Rinobot plugins for automatically processing, plotting and converting data on the fly.

    • [Anaconda python downloads](https://www.continuum.io/downloads Install the distribution for python3.5.

You're all set! Head over to the Getting started guide so start using Rinobot.
